knowledge-clustering

Summary: A command-line assistance tool to facilitate the use of the knowledge LaTeX package.

knowledge-clustering is a command-line tool to help with the use of the knowledge LaTeX package.

In short, knowledge is a package that allows you to easily insert internal hyperlinks from a technical term to its definition. This is particularly useful in scientific papers. Syntax wise, its use is quite please: it suffices to put double quotes around a term to make it clickable, or to use two double quotes to indicate that this is where the term is defined. For instance, if you write "algebra", this will put a link to the paragraph of the document where you’ve written ""algebra"".

The only time-consuming task in using knowledge is that you explicitly have to define synonyms: for instance, you would also like "algebras" to the notion ""algebra"". My tool knowledge-clustering is precisely here to address this issue: use quotes as you wish in your LaTeX document, and then run the tool. It will automatically detect notions that should be considered as synonyms (for instance algebra and algebras, compact and non-compactness), and will propose you to define them as such. This tool is based on a clustering algorithm defined for the occasion, and relies on natural language processing techniques.
